# What a Hugging Face Space installs. It cannot install this project itself:
# Spaces mount only this file into the build and run pip before the repo exists,
# so a `.` line resolves to an empty directory and fails. The deploy therefore
# stages the package at the Space's root, where Python imports it without an
# install, and this file lists what that package needs at runtime.
#
# Only the app's own import path (graph7ph.app -> db, explore, query, render) is
# needed here. pydantic and requests belong to fetch and build, which the Space
# never runs. Versions are exact and must move with uv.lock: a Kùzu release can
# change the on-disk storage format, and the app has to read an artifact the same
# version built. gradio matches the version the Space card declares as its SDK.
kuzu==0.7.1
gradio==5.50.0
pyvis==0.3.2
